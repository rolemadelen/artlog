# Art Log

This is an application to log my daily outfit drawings.

![Demo](./demo.gif)

## Tech Stack
- Database
  + MongoDB
    * Mongoose - `mongoose: "^6.9.2"`
- Client
  + React - `react: ^18.2.0`
  + State Management
    * Redux Toolkit - `@reduxjs/toolkit: ^1.9.2`
  + Sass - `sass: ^1.58.2`
  + Icons from Lottie - `@iconscout/react-unicons: ^1.1.6`,
  + Axios - `axios: ^1.3.3`
- Server
  + Express - `express: ^4.18.2`
